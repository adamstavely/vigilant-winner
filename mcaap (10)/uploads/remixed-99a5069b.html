Design Principles
Clarity over decoration
Every element earns its place. If it doesn't communicate, it doesn't belong.
Efficiency over novelty
Familiar patterns reduce cognitive load.
Confidence over friendliness
A professional tool used by people making real decisions.
Strong information hierarchy

AI & Agents

AI Surface Components

New component patterns for interfaces that contain, generate, or are operated by AI agents. Every pattern is attribution-first — users and other agents always know when content is AI-generated.

Streaming Text

For LLM output that appears progressively. The blinking cursor communicates that the response is live. Uses aria-live="polite" so screen readers and agent parsers track updates in real time.

AI Content Block

A full container for AI-generated content. The accent top border and subtle blue header background distinguish it from standard cards. Accept, Edit, and Regenerate actions give users control. The data-ai-generated attribute makes it machine-readable for other agents parsing the DOM.

AI-generated summary

Revenue grew 12.4% month-over-month to $48,291, driven by three enterprise account expansions in the financial services vertical. This is the strongest Q1 performance since 2023.

Active user count reached 1,284, up 8.1%. Churn held at 2.3% — below the 3% internal target for the third consecutive month.

Recommended action: investigate the conversion rate dip from 9.4% → 8.7% observed after the March 14th onboarding change. Early signals suggest a friction point at the workspace setup step.

Agent-Readable Data Attributes

Standard data-* attributes that every agent-interactive element should carry. These allow AI agents operating the UI to understand intent, risk level, and content provenance without visual inference.

/* Action intent */
data-action="[verb]"               // "save" | "delete" | "export" | "invite"
data-destructive="true"              // flags irreversible consequences
data-requires-confirmation="true"   // agent must confirm before executing

/* Content provenance */
data-ai-generated="true"            // content created by an AI agent
data-source="[system-name]"         // origin system of displayed data
data-last-updated="2026-06-12T..."  // ISO 8601 refresh timestamp
